# 📄 Smart Resume Analyzer (Frontend Only)

A simple and smart **resume analysis web app** built using **HTML, CSS, and JavaScript**.  
This project simulates an **ATS (Applicant Tracking System)** by comparing resume content with predefined job role skills and generating a match score — all on the frontend without any APIs or backend.

---

## 🚀 Features

- Paste or type resume content
- Select predefined job roles
- Skill matching using JavaScript logic
- ATS-style match percentage score
- Matched & missing skills display
- Responsive and clean UI
- No backend, no APIs

---

## 🎯 How It Works

1. User enters resume text
2. Selects a job role
3. JavaScript compares resume text with required skills
4. Calculates an ATS match score
5. Displays matched and missing skills
